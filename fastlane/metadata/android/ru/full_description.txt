С помощью анимированных штрихкодов можно передавать (принимать) данные, используя только камеру. CameraFileCopy (CFC) -- декодер (приёмник) таких кодов. Энкодер (передатчик) доступен по адресу <a href='https://cimbar.org' target='_blank' rel='nofollow'>cimbar.org</a>.

Это односторонний канал передачи. Приложение не использует никакие антенны (Wi-Fi, Bluetooth, NFC и т.п.) или иные «уловки». Благодаря этому оно отлично работает даже в режиме полета.

CFC считывает анимированные <a href='https://github.com/sz3/libcimbar' target='_blank' rel='nofollow'>cimbar-коды</a>. Почти вся логика работы реализована в библиотеке <code>libcimbar</code>, включённой в проект через <code>git-subtree</code>. Сайт <a href='https://cimbar.org/' target='_blank' rel='nofollow'>cimbar.org</a> как компонент-передатчик использует кодировщик <code>cimbar</code>. Чтобы передать файл, откройте этот сайт, выберите файл для передачи, и направьте камеру устройства с запущенным приложением на анимированный штрихкод. Как альтернативный вариант, вы можете сгенерировать коды локально путём использования <code>cimbar_send</code>, утилитой из пакета <code>libcimbar</code>.
